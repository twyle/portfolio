About

I build autonomous AI agents that work seemlessly with humans to accomplish various tasks. This includes the data pipelines that source data used by the ai agents, the tools that the agents use to accomplish various tasks as well as API wrappers that are used by the tools to interact with various services. My proficiency lies in large language models, including prompt engineering as well as natural language processing tasks such named entity recognition, text classification, word embeddings and sentiment analysis. The frameworks and libraries I regurlaly use include langchain, llamaindex, openai, natural language processing toolkit, spacy, gensim, quadrant, scikit-learn and the transformers library.

I currently work at Oryks Systems as a software engineer, building an AI assistant that helps the user work with the youtube API by simply chatting with it. This includes tasks such as creating resources on youtube (uploading videos, creating playlists, creating and answering comments), updating resources ( inserting videos into existing playlists, updating comments) and getting resources (such as live events, searching for channels, videos and playlists). My tasks include building AI agents that leverage various Google services and large language models to enable a user to seemlessly work with the YouTube API. This involves building tools that use various python libraries to interact with the various apis (drive, calendar, sheets, gmail). I also build ETL pipelines especially those that involve scrapping websites for textual data, preprocess the data and store them to various locations including elasticsearch and PostgreSQL. I also write documentation for our users as well as internal docs for new hires and other employees.

I studied Mechatronics Engineering, focusing more on automation.I did component design using Solid Works and modelling using Matlab.

The most interesting project I have worked on is the youtube assistant. This is a chatbot that helps the user accomplish various tasks on youtube such as uploading and downloading videos, commenting on videos and answering comments, creating and updating playlists over a chat-based interface. The aim was to come up with a chat-bot that enabled the user have a conversation with youtube. It uses chat-gpt to understand the user request and translates this into commands that the youtube api can execute. Other projects include a python library that wraps around the youtube api and a social media application that enables the user to share images.

I love working in fast-paced challenging environments especially where data sourcing and processing is involved. My other tech skills includes Amazon Web Services, Linux operating system, Postgres Database and web scraping, Natural language processing. I especially enjoy analyzing textual data.
