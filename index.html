About

I have worked on a wide set of tasks in the natural language processing field including topic modelling especially for reviews, text classification, sentiment analysis, tokenization, named entity recognition, parts of speech tagging, text feature extraction as well as text annotation. I am proficient in web scraping and have built various pipelines that scrape, preprocess and store data from various sites into various data storage solutions such as document databases (mongodb), sql databases (PostgreSQL), search engines(Elasticsearch) and graph databases such as Neo4j. I have also worked with word and sentence embeddings, including huggingface transformers from huggingface and openai embeddings as well as vector stores such as Quadrant, Weaveit and Redis. I do data analysis, especially text analytics and my tools of choice include nltk, spacy, transformers, numpy, pandas, scikit-learn, pyldavis, scrapy, langchain and llamaindex.

I've worked on numerous projects involving natural language processing. My favorite and most challenging so far has been building a chat agent that helps youtubers manage their youtube channel as well as give regular youtube users a better usage experience. The chatbot enables the user to search youtube for videos, playlists, channels, create playlists and add videos to those playlists, view video comments, comment on videos, update and delete comments, answer comments, get channel and video details, update chanel and video metadata, like videos and so much more. Another interesting project is a python library that generates documentation for python projects. Both projects are built with langchain and openai chat-gpt model.

Another interesting project is a machine learning pipeline for training and deploying classification models. This pipeline takes in a datasource, creates a custom data store for storing the raw data including its metadata, generates features, both numerical and categorical using scikit-learn pipelines and transformers and stores the features for later use. It then creates and trains various classification models, picks the best three, finetunes them, then finally picks the best perfoming classifier and deploys this model. It also notifies the user over email of its progress. It's built with python, FastAPI, redis, celery, scikit-learn, docker and PostgreSQL.

I've also built custom python libraries for working with various youtube APIs including YouTube, Gmail and Calendar. These libraries are deployed to pypi and are completely open-source. I've also built a FastAPI app for keeping track of youtube influencers from kenya that runs a cron job that periodically updates it storage using data from the youtube api.

I love working in fast-paced challenging environments especially where data sourcing and processing is involved. I currently work at a startup that I founded that is building AI agents that work with humans to accomplish various tasks as well as the tools that can be used by those ai agents. My focus is on tools that help AI agents work with various Google APIs such as Gmail, YouTube, Calendar, Photos, Sheets, Docs, Gmap and GSlides.
